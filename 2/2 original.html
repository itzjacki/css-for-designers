Del 2 - Boksmodellen

Denne delen handler om å forstå den litt snodige boksmodellen, som selv mange webutviklere aldri har tatt seg tiden til å lære seg skikkelig

Det finnes flere måter å posisjonere disse boksene på, men i denne delen av workshoppen er det kun lov til å flytte boksene med propertiesa som inngår i boksmodellen. Det er greit å bruke andre properties (slik som background-color) dersom det er nødvendig for å etterligne fasit. Man bør nok for eksempel starte med å passe på at boksene er store nok til å være synlige.
